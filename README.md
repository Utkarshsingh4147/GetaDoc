<h2 id="overview">Overview</h2>
<p>GetaDoc is a full-stack healthcare platform that streamlines appointment scheduling, user management, and role-based access control for patients, doctors, and administrators. Built with a modular architecture, it offers a scalable foundation for developing secure and feature-rich healthcare applications.</p>
